# AUR
Some packages that later i'll upload to AUR

# How to install
```bash
git clone https://github.com/propilideno/AUR
cd AUR/PACKAGE_NAME
```
Read the PKGBUILD and ensure that you trust the upstream, then RUN
```bash
makepkg
sudo pacman -U PACKAGE_NAME.pkg
```
